# 🎬 Netflix Data Analysis Project

## 📌 Problem Statement
Analyze Netflix’s content dataset to understand trends in content distribution, genre popularity, and global expansion strategy.

---

## 🛠 Tools Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  

---

## 🧹 Data Cleaning
- Removed duplicate records  
- Handled missing values  
- Converted `date_added` to datetime format  
- Dropped rows with missing date values  

---

## 🔧 Feature Engineering
- Extracted `year_added` and `month_added`  
- Split and exploded `listed_in` for genre analysis  
- Split and exploded `country` for country-level insights  

---

## 📊 Key Insights
- Netflix has approximately **2.4× more movies than TV shows**  
- Major growth observed between **2015–2019**  
- Peak year:
  - 🎬 Movies → **2019**
  - 📺 TV Shows → **2020**  
- **International Movies** is the most dominant genre  
- **United States** ranks first in content production  
- **India ranks second**, highlighting strong regional growth  

---

## 📈 Visualizations
- Content growth over time  
- Movies vs TV Shows comparison  
- Genre distribution  
- Country distribution  

---

## 📌 Conclusion
Netflix’s content strategy reflects strong global expansion, with increasing focus on international markets. Movies dominate the platform, while TV shows show steady growth over time.
